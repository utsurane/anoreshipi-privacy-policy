プライバシーポリシー

最終更新日：2026年6月12日

アノレシピ（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報を適切に管理します。本ポリシーは、本アプリが収集する情報、その利用方法、および保護について説明します。

1. 収集する情報

本アプリは以下の情報を収集します。

アカウント情報

  • メールアドレス（アカウント作成・認証に使用）
  • パスワード（暗号化されてAWS Cognitoで管理）
  • Apple ID（Sign in with Appleを使用した場合）
  • 多要素認証（TOTP）の設定情報（設定した場合。AWS Cognitoで管理）

音声データ

  • レシピ登録時に録音した音声（WAV形式）

レシピ・メモデータ

  • 音声から生成されたレシピの内容

デバイス情報

  • APNsデバイストークン（プッシュ通知の受信・データ同期に使用）

2. 情報の利用目的

収集した情報は以下の目的にのみ使用します。

  • アカウントの認証・管理
  • 音声をテキストに変換し、レシピを生成すること
  • レシピ生成完了のプッシュ通知の送信とレシピデータの同期
  • アプリの機能提供・改善

3. AI処理について

本アプリはレシピ生成にAI技術を使用しています。録音した音声はバックエンドサーバーを経由して以下の流れで処理されます。

音声処理パイプライン

  1. バックエンドサーバーで音声区間検出（VAD）処理
  2. OpenAI Whisper APIで音声をテキストに変換
  3. Anthropic Claude APIでテキストからレシピを生成

OpenAI Whisper API

音声をテキストに変換するために使用します。送信された音声データはOpenAIのプライバシーポリシーに従い処理されます。

Anthropic Claude API

テキストからレシピを生成するために使用します。送信されたテキストはAnthropicのプライバシーポリシーに従い処理されます。

AI処理への同意はアプリ内で確認を行っており、同意なしに処理は行われません。

録音した音声は上記の外部サービスに送信されるため、氏名・住所・電話番号などの個人情報や機密情報は録音に含めないようご注意ください。録音内容に含めた情報の取り扱いについては、ユーザーの責任となります。

4. 情報の保管・管理

  • アカウント情報：AWS Cognitoで管理・暗号化
  • 音声データ：録音時に端末内へ一時保存され、アップロード完了後に端末から削除されます。サーバー上の音声はAI処理完了後にAWS S3から削除されます。削除に失敗した場合は最長24時間以内に自動削除されます
  • レシピデータ：クラウドデータベース（Turso DB）に保存されます。端末内には永続保存せず、アプリ利用中のみメモリ上に保持します。アカウント削除時に削除されます
  • ジョブ管理データ（処理の進行状況）：AWS DynamoDBに保存し、24時間後に自動削除されます
  • APNsデバイストークン：AWS DynamoDBに保存。アカウント削除時に削除されます
  • 認証トークン：端末のKeychainに暗号化して保存
  • APIアクセスログ（リクエスト元IPアドレス・HTTPメソッド・パス・ステータスコードを含む）：AWS CloudWatchに90日間保持後、自動削除されます

AWS上で管理するデータは、AWSの東京リージョン（ap-northeast-1）で処理・保管されます。外部サービスで扱われるデータは、各サービスの方針に従って管理されます。

5. 安全管理

運営者は、通信および保存データの暗号化など、合理的な範囲で個人情報の安全管理に努めます。

万一、個人情報の漏洩等が発生した場合は、法令に従い必要な対応を行います。

6. 第三者への提供

収集した個人情報は、以下の場合を除き第三者に提供しません。

  • ユーザーの同意がある場合
  • 法令に基づく開示が必要な場合
  • AI処理のために必要な範囲でOpenAI・Anthropicに送信する場合（第3条参照）

7. データの削除

ユーザーはいつでも本アプリからアカウントを削除できます。アカウントを削除すると、以下のデータが削除されます。

  • アカウント情報（認証情報）
  • 端末内の認証トークン
  • クラウドデータベースに保存されたすべてのレシピデータ（端末のメモリ上に保持中のデータも消去されます）
  • 端末内に一時保存された録音済み音声ファイル
  • バックエンドサーバーに登録されたAPNsデバイストークン

アカウント削除は設定メニュー（画面左上の人物アイコン）から「アカウントを削除」を選択して実行できます。

8. マイクの使用

本アプリはレシピを音声で登録するためにマイクを使用します。マイクへのアクセスはレシピ録音時のみ行われます。録音中にアプリをバックグラウンドへ切り替えた場合や画面をロックした場合は、録音を停止するまで録音が継続します。録音していない状態でバックグラウンドから録音を開始することはありません。

9. 第三者の音声の録音について

レシピの録音時に、ユーザーの周囲にいる方の声が音声に含まれる場合があります。

ユーザーは、録音を行う際、周囲の方に録音する旨をあらかじめ知らせ、必要な同意を得る責任を負います。第三者の声が含まれる音声の取り扱いについては、ユーザーの責任において行ってください。

録音に起因して生じた第三者のプライバシー・個人情報に関する紛争または損害について、運営者は、運営者の故意または重大な過失による場合を除き、責任を負いません。

10. プッシュ通知

本アプリはレシピ生成の完了を通知するためにプッシュ通知を使用します。通知を受信すると、バックグラウンドでクラウドデータベースと自動同期し、最新のレシピデータを取得します。プッシュ通知の受信はOS設定からいつでも無効化できます。

11. プライバシーポリシーの変更

本ポリシーの内容は、法令の改正や機能の追加・変更に応じて改定されることがあります。重要な変更を行う場合は、アプリ内および本ページでお知らせします。改定後のポリシーは、掲載した時点から効力を生じます。
