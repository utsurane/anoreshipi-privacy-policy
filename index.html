プライバシーポリシー

最終更新日：2026年5月11日

アノレシピ（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報を適切に管理します。本ポリシーは、本アプリが収集する情報、その利用方法、および保護について説明します。

1. 収集する情報

本アプリは以下の情報を収集します。

アカウント情報

  • メールアドレス（アカウント作成・認証に使用）
  • パスワード（暗号化されてAWS Cognitoで管理）

音声データ

  • レシピ登録時に録音した音声（WAV形式）

レシピ・メモデータ

  • 音声から生成されたレシピの内容

2. 情報の利用目的

収集した情報は以下の目的にのみ使用します。

  • アカウントの認証・管理
  • 音声をテキストに変換し、レシピを生成すること
  • アプリの機能提供・改善

3. AI処理について

本アプリはレシピ生成にAI技術を使用しています。録音した音声は以下の外部サービスに送信されます。

OpenAI Whisper API

音声をテキストに変換するために使用します。送信された音声データはOpenAIのプライバシーポリシーに従い処理されます。

Anthropic Claude API

テキストからレシピを生成するために使用します。送信されたテキストはAnthropicのプライバシーポリシーに従い処理されます。

AI処理への同意はアプリ内で確認を行っており、同意なしに処理は行われません。

4. 情報の保管・管理

  • アカウント情報：Amazon Web Services（AWS）Cognitoで管理・暗号化
  • 音声データ：処理完了後にAWS S3から削除されます
  • レシピデータ：AWS S3に保存され、アカウント削除時に削除されます
  • 認証トークン：端末のKeychainに暗号化して保存

すべてのデータはAWSの東京リージョン（ap-northeast-1）に保存されます。

5. 第三者への提供

収集した個人情報は、以下の場合を除き第三者に提供しません。

  • ユーザーの同意がある場合
  • 法令に基づく開示が必要な場合
  • AI処理のために必要な範囲でOpenAI・Anthropicに送信する場合（第3条参照）

6. データの削除

ユーザーはいつでも本アプリからアカウントを削除できます。アカウントを削除すると、以下のデータが削除されます。

  • Cognitoアカウント情報
  • 端末内の認証情報（Keychain）

アカウント削除は設定メニュー（画面左上の人物アイコン）から「アカウントを削除」を選択して実行できます。

7. マイクの使用

本アプリはレシピを音声で登録するためにマイクを使用します。マイクへのアクセスはレシピ録音時のみ行われ、バックグラウンドでの録音は行いません。
